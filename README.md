# 📍 Localizador de Casas de Amor

🔗 Encontre rapidamente a Casa de Amor mais próxima de você com apenas um clique.

---

## 🚀 Sobre o projeto

O **Localizador de Casas de Amor** é uma aplicação web simples e eficiente que utiliza a geolocalização do usuário para identificar automaticamente a unidade mais próxima e fornecer uma rota direta via Google Maps.

O projeto foi pensado para oferecer praticidade no convite de novos participantes, eliminando a necessidade de enviar endereços manualmente.

## 🧠 Como funciona

* O usuário acessa o site
* Autoriza o uso da localização
* O sistema calcula a distância entre o usuário e os locais cadastrados
* Exibe o local mais próximo
* Permite abrir a rota diretamente no Google Maps

---

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **JavaScript (Vanilla)**
* **Geolocation API**
* Integração com Google Maps

## ⚙️ Funcionalidades

✔️ Localização automática do usuário
